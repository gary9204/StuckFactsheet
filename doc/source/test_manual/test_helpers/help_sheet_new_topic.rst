Helper Factsheet New Topic
==========================

This helper provides a checklist to add a topic to a factsheet.  The
steps assume you have created or opened a factsheet in the application.

.. include:: /icons/icons-include.txt

1. Select *Topics* **New Topic icon** |document-new| (*Topics* pane
   toolbar, second icon from left).

   *Place New Topic* dialog appears.

   .. note:: The application skips the *Place New Topic* dialog for the
      first topic you add to the outline.

#. Select sibling (parent) topic for new topic (*Place New Topic* dialog
   *Topics* pane).

#. Click positiion button **After** (respectively, **Child**) for the
   order of the new topic.

#. Click **Place button** (dialog title far right).

   *Select Template* dialog appears.

#. Click on template ``name_0xx`` in *Templates* pane of dialog.

#. Click **Specify button** (*Select Template* dialog title far right).

#. Click **Next button** (*New Section* dialog title far right).

#. Click in **Name** field and type topic name.

#. Click in **Title** field and type topic title.

#. Click in **Description** field and type topic description.

#. Click **Next button**.

#. Click **Apply button** (dialog title far right).

   Control returns to the factsheet window.

The *Topics* pane should now list a new topic with name and title
entered above.
