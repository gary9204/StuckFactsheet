Case Factsheet Dialogs¶

Purpose: demonstrate factsheet dialogs Help, Display Help, and Data Loss Warning.

Setup¶

None

Steps – Help¶

  1. Step: Start application with Helper Start application.

    1. Expect: Application displays default factsheet.

  2. Step: Click Factsheet menu item Help (right of Factsheet field).

    1. Expect: Factsheet Help dialog appears.

  3. Step: Review dialog contents (title, text, references, etc.).

    1. Expect: Content is current and correct.

  4. Step: Click dialog close icon (dialog title far right).

    1. Expect: Dialog disappears.

Steps – Display Help¶

  1. Step: Click Factsheet menu item Display … > Help (right of Factsheet field).

    1. Expect: Factsheet Display Help dialog appears.

  2. Step: Review dialog contents (title, text, references, etc.).

    1. Expect: Content is current and correct.

  3. Step: Click dialog close icon (dialog title far right).

    1. Expect: Dialog disappears.

Steps – File Help¶

  1. Step: Click Factsheet menu item File … > Help (right of Factsheet field).

    1. Expect: Factsheet File Help dialog appears.

  2. Step: Review dialog contents (title, text, references, etc.).

    1. Expect: Content is current and correct.

  3. Step: Click dialog close icon (dialog title far right).

    1. Expect: Dialog disappears.

Steps – Topics Help¶

  1. Step: Click Topics menu item Help (pane toolbar, last icon on right).

    1. Expect: Factsheet Topics Help dialog appears.

  2. Step: Review dialog contents (title, text, references, etc.).

    1. Expect: Content is current and correct.

  3. Step: Click dialog close icon (dialog title far right).

    1. Expect: Dialog disappears.

Steps – Data Loss Warning¶

  1. Step: Type text in the Factsheet field (e.g. ‘Sample’).

    1. Expect: Text appears in field.

  2. Step: Click window close icon (window title far right).

    1. Expect: Data Loss Warning dialog appears.

  3. Step: Review dialog contents (title, text, buttons, etc.).

    1. Expect: Content is current and correct.

  4. Step: Click Discard button (dialog title far right).

    1. Expect: Dialog disappears and application closes.

Teardown¶

  1. Check console for exceptions, GTK errors, and warning messages. There should be none.
