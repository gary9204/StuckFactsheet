Case Factsheet Save and Open¶

Purpose: demonstrate Factsheet save to file and open from file.

Setup¶

  1. Create a test data directory (for example, /home/Scratch/factsheet_test).

  2. In the test data directory create a file test.txt containing a short text string (e.g., “The Spanish Inquisition.”

Steps – Save and Open¶

  1. Step: Start application with Helper Start application

    1. Expect: Application displays default factsheet.

  2. Step: Edit ( icon on left) factsheet name “Unnamed” to “Alpha”.

    1. Expect: “Alpha” appears in field.

  3. Step: Edit ( icon on right) factsheet title to “First factsheet”.

    1. Expect: “First factsheet” appears in field.

  4. Step: Click inside the Factsheet Summary frame after existing text.

    1. Expect: A blinking text cursor (|) appears after existing text.

  5. Step: Edit the summary to “This is the first factsheet.”

    1. Expect: The text appears as entered.

  6. Step: Click Factsheet Save button (window title on right near center).

    1. Expect: Save dialog appears.

    2. Expect: Dialog file Name field contains “factsheet.fsg”.

  7. Step: Change to test data directory.

    1. Expect: Save dialog shows directory contents.

  8. Step: Edit file Name field to “Alpha.fsg”

    1. Expect: Field contains new file name.

  9. Step: Click Save button (dialog title on right).

    1. Expect: Dialog disappears.

  10. Step: List test data directory contents (in shell or file browser).

    1. Expect: File Alpha.fsg appears with current time stamp.

    2. Take note of file time and size.

  11. Step: Click window close icon (window title far right).

    1. Expect: Window disappears.

    2. Expect: Application closes.

  12. Step: Start application with Helper Start application

    1. Expect: Application displays default factsheet.

  13. Step: Click Factsheet Open button (window title on far left).

    1. Expect: Open dialog appears.

  14. Step: Change to test data directory.

    1. Expect: Open dialog shows directory contents.

  15. Step: Select Alpha.fsg and click Open button (dialog title on right)

    1. Expect: Open dialog disappears.

    2. Expect: Second window appears with window title “Alpha”.

    3. Expect: Name is “Alpha”.

    4. Expect: Title is “First factsheet”.

    5. Expect: Summary contains “This is the first factsheet.”

    6. New window may cover first window.

  16. Step: Edit Alpha factsheet name to “Alpha Plus”.

    1. Expect: Updated title appears in field.

  17. Step: Edit the summary to “This is the first factsheet with additions.”

    1. Expect: The text appears as entered.

  18. Step: Select Factsheet menu item File … > Save. ( icon on far right of Factsheet identity line).

    1. Expect: Menu disappears.

  19. Step: List test data directory contents.

    1. Expect: File Alpha.fsg appears with updated size and time stamp.

    2. Expect: File Alpha.fsg~ appears with size and time stamp of initial Alpha.fsg file.

  20. Step: Click window close icon .

    1. Expect: Window disappears.

  21. Step: In original window, click window close icon .

    1. Expect: Window disappears.

    2. Expect: Application closes.

  22. Step: Start application with Helper Start application

    1. Expect: Application displays default factsheet.

  23. Step: Select Factsheet menu item File … > Open ….

    1. Expect: Open dialog appears.

    2. Expect: File filter button in lower right corner is labeled “Factsheet”.

  24. Step: In dialog, change to test data directory.

    1. Expect: Open dialog shows directory contents.

  25. Step: Click filter button.

    1. Expect: List of file filters pops up: “Factsheet” and “Any”.

  26. Step: Click “Any” filter.

    1. Expect: Pop up collapses to “Any” on filter button.

    2. Expect: File list includes test.txt.

  27. Step: Click filter button.

    1. Expect: List of file filters pops up: “Factsheet” and “Any”.

  28. Step: Click “Factsheet” filter.

    1. Expect: Pop up collapses to “Factsheet” on filter button.

    2. Expect: File list only includes files with fsg extension.

  29. Step: Select Alpha.fsg and click Open button.

    1. Expect: Open dialog disappears.

    2. Expect: Second window appears with window title “Alpha Plus”.

    3. Expect: Name is “Alpha Plus”.

    4. Expect: Title is “First factsheet”.

    5. Expect: Summary contains “This is the first factsheet with additions.”

    6. New window may cover first window.

  30. Step: Click window close icon .

    1. Expect: Window disappears.

  31. Step: In original window, click window close icon .

    1. Expect: Window disappears.

    2. Expect: Application closes.

Steps – Save as¶

  1. Step: Start application with Helper Start application

    1. Expect: Application displays default factsheet.

  2. Step: Edit ( icon on left) factsheet name “Unnamed” to “Beta”.

    1. Expect: “Beta” appears in field.

  3. Step: Edit ( icon on right) factsheet title to “Second factsheet”.

    1. Expect: “Second factsheet” appears in field.

  4. Step: Click inside the Factsheet Summary frame after existing test.

    1. Expect: A blinking text cursor (|) appears after existing text.

  5. Step: Edit the summary to “This is the second factsheet.”

    1. Expect: The text appears as entered.

  6. Step: Click Factsheet Save button (window title on right near center).

    1. Expect: Save dialog appears.

    2. Expect: Dialog file Name field contains “factsheet.fsg”.

  7. Step: Change to test data directory.

    1. Expect: Save dialog shows directory contents.

  8. Step: Edit file Name field to “Beta.fsg”

    1. Expect: Field contains new file name.

  9. Step: Click Save button (dialog title on right).

    1. Expect: Dialog disappears.

  10. Step: List test data directory contents.

    1. Expect: Files Alpha.fsg and Beta.fsg appear is list.

  11. Step: Edit Beta factsheet name to “Gamma”.

    1. Expect: Updated name appears in field.

    2. Expect: Window title changes to “Gamma”.

  12. Step: Edit Gamma factsheet title to “Third factsheet”.

    1. Expect: Updated title appears in field.

  13. Step: Edit the summary to “This is the third factsheet.

    1. Expect: The text appears as entered.

  14. Step: Click Factsheet Save as icon (window title on right near Save button).

    1. Expect: Save dialog appears.

    2. Expect: Dialog file Name field contains “Beta.fsg”.

  15. Step: Change to test data directory.

    1. Expect: Save dialog shows directory contents.

  16. Step: Edit file Name field to “Gamma.fsg”

    1. Expect: Field contains new file name.

  17. Step: Click Save button.

    1. Expect: Dialog disappears.

  18. Step: List test data directory contents.

    1. Expect: Files Alpha.fsg, Beta.fsg, and Gamma.fsg appear is list.

  19. Step: Select Factsheet menu item File … > Save as. ( icon on far right of Factsheet identity line).

    1. Expect: Save dialog appears.

    2. Expect: Dialog file Name field contains “Gamma.fsg”.

  20. Step: Change to test data directory.

    1. Expect: Save dialog shows directory contents.

  21. Step: Edit file Name field to “Delta.fsg”

    1. Expect: Field contains new file name.

  22. Step: Click Save button.

    1. Expect: Dialog disappears.

  23. Step: List test data directory contents.

    1. Expect: Files Alpha.fsg, Beta.fsg, Gamma.fsg, and Delta.fsg appear is list.

    2. Expect: Files Gamma.fsg and Delta.fsg have the same size.

  24. Step: in Delta window, click window close icon .

    1. Expect: Window disappears.

    2. Expect: Application closes.

Steps – Duplicate Open¶

  1. Step: Start application with Helper Start application

    1. Expect: Application displays default factsheet.

  2. Step: Click Factsheet Open button (window title on far left).

    1. Expect: Open dialog appears.

  3. Step: Change to test data directory.

    1. Expect: Open dialog shows directory contents.

  4. Step: Select Beta.fsg and click Open button (dialog title on right)

    1. Expect: Open dialog disappears.

    2. Expect: Second window appears with window title “Beta”.

    3. Expect: Name is “Beta”.

    4. Expect: Title is “Second factsheet”.

    5. Expect: Summary contains “This is the second factsheet.”

    6. New window may cover first window.

  5. Step: Select Factsheet menu item Display … > Open window ( icon on far right of Factsheet identity line).

    1. Expect: New Beta window appears.

    2. New window may cover existing windows.

  6. Step: Position windows so that Unamed window covers parts of both Beta windows.

    1. Expect: Unnamed window obscures parts of Beta windows.

  7. Step: Click Factsheet Open button.

    1. Expect: Open dialog appears.

  8. Step: Change to test data directory.

    1. Expect: Open dialog shows directory contents.

  9. Step: Select Beta.fsg and click Open button.

    1. Expect: Open dialog disappears.

    2. Expect: Both Beta windows cover parts of Unnamed window.

    3. Expect: No new windows appear.

  10. Step: In Beta window, select Factsheet menu item File … > Close. ( icon on far right of Factsheet identity line).

    1. Expect: Both Beta windows disappear.

  11. Step: click Unnamed window close icon .

    1. Expect: Unnamed window disappears.

    2. Expect: application closes.

Steps – Data Loss and Overwrite Warnings¶

  1. Step: Start application with Helper Start application

    1. Expect: Application displays default factsheet.

  2. Step: Edit ( icon on left) factsheet name “Unnamed” to “<b>Guards</b>”.

    1. Expect: “Guards” appears in field.

    2. Expect: “Guards” appears in window title.

  3. Step: Click Factsheet Save button (window title on right near center).

    1. Expect: Save dialog appears.

    2. Expect: Dialog Name field contains “factsheet.fsg”.

  4. Step: Change to test data directory.

    1. Expect: Save dialog shows directory contents.

  5. Step: Edit file Name to “guards.fsg”.

    1. Expect: Field contains new file name.

  6. Step: Click Save button (dialog title on right).

    1. Expect: Dialog disappears.

  7. Step: List test data directory contents (in shell or file browser).

    1. Expect: File guards.fsg appears with current time stamp.

  8. Step: Edit ( icon on right) factsheet title to “Guard factsheet”.

    1. Expect: “Guard factsheet” appears in field.

  9. Step: Click window close icon (window title far right).

    1. Expect: Data Loss Warning dialog appears.

    2. Expect: First warning line is: “Factsheet Guards contains unsaved changes. All unsaved changes will be discarded if you close.”

    3. Expect: Second warning line is: “Cancel close, or continue to close and discard changes?”

  10. Step: Click Cancel button

    1. Expect: Dialog disappears.

  11. Step: In the test data directory create a file target.fsg containing a short test string (e.g., “And now the comfy chair!”

    1. Expect: Test data directory contains file target.fsg.

  12. Step: Click window Save as icon (window title to right of Save button).

    1. Expect: Save dialog appears.

    2. Expect: Dialog Name field contains “guards.fsg”.

  13. Step: Select target.fsg and click Save button (dialog title on right)

    1. Expect: Dialog appears warning file already exists.

  14. Step: Click Cancel button.

    1. Expect: Warning dialog disappears.

    2. Expect: Save dialog remains visible.

  15. Step: List contents of target.fsg

    1. Expect: Contents unchanged.

  16. Step: Select target.fsg and click Save button.

    1. Expect: Dialog appears warning file already exists.

  17. Step: Click Replace button.

    1. Expect: Warning dialog and save dialog disappear.

  18. Step: From console, open file target.fsg in hex editor (e.g. ghex).

    1. Expect: File contains binary pickle data.

    2. Close hex editor after check.

  19. Step: From console, display contents with python3 -m pickletools target.fsg.

    1. Expect: Output includes factsheet class names.

    2. Expect: Output includes factsheet name and title text.

    3. Return to application after check.

  20. Step: Click window close icon .

    1. Expect: Window disappears.

    2. Expect: Application closes.

  21. Step: Delete file target.fsg.

    1. Expect: Test data directory does not contain the file.

Teardown¶

  1. Delete test data directory along with its contents.

  2. Check console for exceptions, GTK errors, and warning messages. There should be none.
