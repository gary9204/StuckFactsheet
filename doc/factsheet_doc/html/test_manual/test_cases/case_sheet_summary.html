Case Factsheet Summary¶

Purpose: confirm display and edit of Factsheet summary.

Setup¶

  1. Step: Start application with Helper Start application

Steps – Appearance and Basic Editing¶

  1. Step: Check window layout.

    1. Expect: A frame with collapser icon (▾) and label “Factsheet Summary” appears immediately below the edit icon for the Factsheet name.

    2. Expect: Default text for Factsheet summary is “Edit Factsheet description here.”

  2. Step: Click inside the Factsheet Summary frame after the default text.

    1. Expect: A blinking text cursor (|) appears after the default text.

  3. Step: Edit the summary as follows: “There is a tide in the affairs of men, which taken at the flood, leads on to fortune. Omitted, all the voyage of their lives is bound in shallows and in misery.”

    1. Expect: The text appears as entered.

    2. Expect: The text wraps to fit the frame.

  4. Step: Grab the lower right corner of the window and resize the window.

    1. Expect: Summary text wraps to fit the resized frame.

    2. Expect: Wrapping is at white space between words.

  5. Step: Select and copy summary text. Paste into another application (e.g. Vim).

    1. Expect: Factsheet application highlights text when selected.

    2. Expect: Pasted text matches summary text.

  6. Step: Copy “And now for something completely different!” including a new line and paste at the bottom of the summary.

    1. Expect: Copied text appears in summary and matches original.

  7. Step: Repeat paste about 20 time at the bottom of the summary.

    1. Expect: Copied text appears in summary and matches original.

    2. Expect: Summary pane scrolls as needed to keep each paste visible.

  8. Step: Move scroll slider to review summary content.

    1. Expect: Pane scrolls to cover all summary content.

  9. Step: Lengthen window and move pane divider down to expose entire summary.

    1. Expect: Scrollbar disappears when entire summary is visible.

  10. Step: Move pane divider up as far as possible.

    1. Expect: Pane divider stops with about three summary lines visible.

Steps – Hide/Show Summary¶

  1. Step: Edit summary to “This sample factsheet includes a summary.”

    1. Expect: Text appears in summary pane as typed.

  2. Step: Click collapser icon (▾) to left of Factsheet Summary label.

    1. Expect: The Factsheet Summary pane collapses and shows only expander icon (▸) and label.

    2. Expect: The Factsheet Topics pane expands to replace it.

  3. Step: Click expander icon (▸) to left of Factsheet Summary label.

    1. Expect: The Factsheet Summary pane expands.

    2. Expect: The Factsheet Topics pane contracts to accommodate it.

    3. Expect: The Factsheet Summary text is unchanged.

  4. Step: Select Factsheet menu item Display … > Summary (right of Factsheet name and title fields).

    1. Expect: The Factsheet Summary label, frame, and pane disappear.

    2. Expect: The Factsheet Topics pane expands to fill the window.

    3. Expect: The popup menu remains visible.

    4. Expect: The check box to the left of Summary menu item is cleared.

  5. Step: Select Summary menu item.

    1. Expect: The Factsheet Summary label, frame, and pane appear.

    2. Expect: The Factsheet Topics pane contracts to accommodate them.

    3. Expect: The Factsheet Summary text is unchanged.

    4. Expect: The popup menu remains visible.

    5. Expect: The Summary check box is checked.

  6. Step: Press Esc key.

    1. Expect: The popup menu disappears.

  7. Step: Click collapser icon (▾) to left of Factsheet Summary label.

    1. Expect: The Factsheet Summary pane collapses and shows only expander icon (▸) and label.

    2. Expect: The Factsheet Topics pane expands to replace it.

  8. Step: Select Factsheet menu item Display … > Summary.

    1. Expect: The Factsheet Summary label disappears.

    2. Expect: The Factsheet Topics pane expands to fill the window.

    3. Expect: The popup menu remains visible.

    4. Expect: The check box to the left of Summary menu item is cleared.

  9. Step: Select Summary menu item.

    1. Expect: The Factsheet Summary label appears.

    2. Expect: The Factsheet Topics pane contracts to accommodate them.

    3. Expect: The popup menu remains visible.

    4. Expect: The Summary check box is checked.

  10. Step: Press Esc key.

    1. Expect: The popup menu disappears.

  11. Step: Click initial window close icon .

    1. Expect: Data Loss Warning dialog appears.

  12. Step: Click Discard button.

    1. Expect: Dialog and window disappear.

    2. Expect: Application closes.

Teardown¶

  1. Check console for exceptions, GTK errors, and warning messages. There should be none.

  2. Transient Gdk-WARNING on 2021-06-16

    • Warning: (app.py:10104): Gdk-WARNING …: Error writing selection data: Error writing to file descriptor: Broken pipe

    • Reproduce:

      1. Enter text in Factsheet Summary frame.

      2. Select text with mouse

    • Notes:

      1. I could reproduce the warning on June 16 but not the following day.

      2. On June 16, moving mouse cursor to change size of selection triggers multiple warnings.

      3. Double and triple clicking to select text does not trigger warning.

      4. Selecting text with arrow keys does not trigger warning.

  3. Transient Gdk-WARNING on 2021-06-22 repeated

    • Warning: (app.py:10104): Gdk-WARNING …: Error writing selection data: Error writing to file descriptor: Broken pipe

    • Reproduce:

      1. Enter text in Factsheet Summary frame.

      2. Select text with mouse

    • Notes:

      1. The warning recurred after lunch.

      2. I paused the virtual machine hosting the development environment during lunch.

      3. After rebooting the virtual machine, the warning did not recur.
