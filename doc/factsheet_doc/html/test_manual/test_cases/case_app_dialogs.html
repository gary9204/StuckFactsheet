Case Application Dialogs¶

Purpose: demonstrate application dialogs About, Help, and Introduction.

Setup¶

None

Steps – About¶

  1. Step: Start application with Helper Start application

    1. Expect: Application displays default factsheet.

  2. Step: Click Application menu item About (window title on right).

    1. Expect: Application About dialog appears.

  3. Step: Review dialog contents (title, version, text, etc.).

    1. Expect: Content is current and correct.

  4. Step: Click dialog close icon (dialog title far right).

    1. Expect: Dialog disappears. Factsheet window reamins visible.

Steps – Help¶

  1. Step: Click Application menu item Help.

    1. Expect: Application Help dialog appears.

  2. Step: Review dialog contents (title, text, etc.).

    1. Expect: Content is current and correct.

  3. Step: Click dialog close icon (dialog title far right).

    1. Expect: Dialog disappears. Factsheet window remains visible.

Steps – Introduction¶

  1. Step: Click Application menu item Introduction.

    1. Expect: Application Introduction dialog appears.

  2. Step: Review dialog contents (title, text, etc.).

    1. Expect: Content is current and correct.

  3. Step: Click dialog close icon (dialog title far right).

    1. Expect: Dialog disappears.

  4. Step: Click window close icon (window title far right).

    1. Expect: Window disappears and application closes.

Teardown¶

  1. Check console for exceptions, GTK errors, and warning messages. There should be none.
