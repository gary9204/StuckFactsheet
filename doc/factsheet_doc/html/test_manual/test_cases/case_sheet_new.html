Case Factsheet New¶

Purpose: demonstrate Factsheet creation.

Setup¶

None

Steps¶

  1. Step: Start application with Helper Start application

    1. Expect: Application displays default factsheet.

  2. Step: Edit ( icon on left) factsheet name “Unnamed” to “Alpha”.

    1. Expect: “Alpha” appears in field.

  3. Step: Click Application new Factsheet icon (window title on left next to open buttons).

    1. Expect: Second window appears with name Unnamed.

    2. New window may cover first window.

  4. Step: Edit factsheet name “Unnamed” to “Beta”.

    1. Expect: “Beta” appears in field.

    2. Expect: No change in Alpha window name.

  5. Step: Edit ( icon on right) title in Beta window to “<i>Factsheet</i> Beta”.

    1. Expect: Title changes to “Factsheet Beta” in Beta window.

    2. Expect: Title remains “New Factsheet” in Alpha window.

  6. Step: Select Factsheet menu item File … > new ( icon on far right of Factsheet identity line).

    1. Expect: Third window appears with name Unnamed.

    2. New window may cover other windows.

  7. Step: Edit factsheet name “Unnamed” to “Gamma”.

    1. Expect: “Gamma” appears in field.

    2. Expect: No change in Alpha and Beta window names.

  8. Step: In Alpha window, click window close icon (window title far right).

    1. Expect: Data Loss Warning dialog appears.

  9. Step: Click Discard button (dialog title far right).

    1. Expect: Dialog and Alpha window disappear.

    2. Expect: Beta and Gamma windows remain.

  10. Step: In Gamma window, click window close icon .

    1. Expect: Data Loss Warning dialog appears.

  11. Step: Click Discard button.

    1. Expect: Dialog and Gamma window disappear.

    2. Expect: Beta windows remains.

  12. Step: In window, click window close icon .

    1. Expect: Data Loss Warning dialog appears.

  13. Step: Click Discard button

    1. Expect: Dialog and Beta window disappear.

    2. Expect: Application closes.

Teardown¶

  1. Check console for exceptions, GTK errors, and warning messages. There should be none.
