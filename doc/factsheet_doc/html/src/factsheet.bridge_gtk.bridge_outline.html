bridge_outline Module¶

Defines bridge classes to encapsulate widget toolkit outline classes.

An outline is an ordered list analogous to a bulleted list in a print document. An outline contains lines, each of which identifies an item. Lines and items are analogous to bullets and text in a bulleted list. Like a bulleted list, an outline may be either single-level or multi- level.

Constants and Type Hints¶

ChooserOutline¶

Type hint for GTK element to view an outline and choose a line. See Gtk.ComboBox.

ItemOpaque¶

Placeholder type hint for an item at a line.

LineOutline¶

Type hint for GTK element to store a line. See Gtk.Treeiter.

PersistOutline¶

Type hint for outline representation suitable for persistent storage.

UiModelOutlineMulti¶

Type hint for GTK element to store a multi-level outline. See Gtk.TreeStore.

UIModelOutlineSingle¶

Type hint for GTK element to store a single-level outline. See Gtk.ListStore .

ViewOutline¶

Type hint for GTK element to view an outline in columnar format. See Gtk.TreeView.

insert_section(p_other, p_line_parent=None, p_line_section=None)[source]¶

Insert section into outline from other outline.

Copy section of other outline under given line after all existing children. If given line is None, append section at end of outline. If line in other outline is None, copy all of other outline.

Note

The new section is a shallow copy of the other outline’s section.

Parameters

  • p_other (ModelOutlineMulti) – outline that contains section to copy.

  • p_line_parent (Optional[TreeIter]) – line of parent for section copy. Default is top level after existing top-level items.

  • p_line_section (Optional[TreeIter]) – line to copy along with all descendants. Default is to copy all lines.

Return type

None
