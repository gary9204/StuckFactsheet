view_topic View Module¶

Defines class to display topic view in a Factsheet window pane.

class ViewTopic(*, p_control)[source]¶

Bases: object

Display topic and translate user actions.

Class ViewTopic maintains presentation of a topic in a pane of a Factsheet window. The class displays the content of a topic model. It translates a user’s actions at the user interface into requests to update the model and its presentation.

Note

Enhancement opportunity

Method on_show_dialog includes a top-level window check as per Gtk.Widget.get_toplevel. During normal operation, this check should always confirm the topic pane is in a top-level window.

The method silently patches a failure. Consider adding a log message to the method to document abnormal behavior.
