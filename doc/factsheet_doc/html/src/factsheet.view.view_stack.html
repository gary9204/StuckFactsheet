view_stack View Module¶

Defines classes to represent a collection of views.

Factsheet uses outlines containing topics and facts. ViewStack provides a collection to contain and display views of items in an outline. ViewStack presents one view at a time.

class ViewStack[source]¶

Bases: object

Displays view at a time from a collection of views.

Each view is a presentation element for an item (such as a topic, fact, or fact value). Each item view is identified by a name. Methods use view name to add, show, pin, or remove the item view. The class supports pinning names. When a name is pinned, the corresponding view cannot be removed.

add_view(p_view, p_name)[source]¶

Add an item view with the given name to the collection.

When the collection contains a view with the given name, log a warning and do not change the collection.

Parameters

  • p_name (str) – name of view. A name may appear at most once in the collection.
