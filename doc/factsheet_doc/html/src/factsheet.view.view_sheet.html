new_dialog_warn_loss(p_parent, p_name)[source]¶

Return Data Loss Warning dialog.

Parameters

  • p_parent (ApplicationWindow) – window running dialog.

  • p_name (str) – name of factsheet that might lose data.

Note

There are limitations in Glade and Python bindings for GTK. Glade does not recognize use-header-bar property of GtkDialog. Gtk.Dialog() does not recognize flag Gtk.DialogFlags.USE_HEADER_BAR.

To replace hard-coded ui definition with a Glade file, Manually add the following to GtkDialog section of the Glade file:

<property name=”use-header-bar”>1</property>

Return type

Dialog

class ViewSheet(*, p_control)[source]¶

Bases: factsheet.control.control_sheet.ObserverControlSheet

Displays Factsheet document and translates user actions.

Class ViewSheet maintains presentation of a Factsheet. The class displays the content of a factsheet model. It translates a user’s actions at the user interface into requests to update the model and its presentation.

on_delete_sheet(_action, _target)[source]¶

Delete factsheet guarding against data loss.

A user may ask to delete a factsheet when there are unsaved changes. If so, the method includes checks to ensure the user approves. The method deletes the factsheet unconditionally if no changes would be lost.
