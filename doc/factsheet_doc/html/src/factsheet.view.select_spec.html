class SelectSpec(p_parent)[source]¶

Bases: object

Provides the means to select the specification for a new topic.

Select Specification dialog allows a user to select the specification for a new topic from an outline of available specications. A user may cancel the dialog without selecting a specification.

NO_SUMMARY¶

Summary text displayed when no specification is selected.
