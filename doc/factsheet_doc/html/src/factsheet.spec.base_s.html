base_s Module¶

Defines base classes for topic specification.

class Base(*, p_name, p_summary, p_title)[source]¶

Bases: object

Base spec for Factsheet topics.

A spec provides a user the means to create a class of topics. A spec embodies a template for the class. It queries the user to complete the template for a specific topic. The spec also queries the user of the location of the new topic in the Factsheet’s outline of topics.

__call__()[source]¶

Orchestrate creation and placement of new topic.

The user specializes the specification to create the topic. The user may confirm or cancel the topic.
